# VPS Speedtest Telegram Bot 🚀

Легкий Telegram-бот на Python и `aiogram` для измерения и мониторинга скорости интернета на вашем VPS/сервере.

## Возможности
- **Ручной запуск:** Мгновенное измерение скорости скачивания, загрузки и пинга по команде `/start`.
- **Постоянный мониторинг:** Автоматическое фоновое тестирование каждые 10 минут с отправкой результатов прямо в чат.
- **В контейнере:** Простая и удобная установка с помощью Docker и Docker Compose.

## Требования
- Установленные Docker и Docker Compose на вашем сервере.
- Токен Telegram-бота (можно получить у [@BotFather](https://t.me/BotFather)).

## Установка и запуск

1. Склонируйте репозиторий на ваш сервер:
   ```bash
   git clone <ссылка-на-ваш-репозиторий>
   cd test-vps-bot
   ```

2. Откройте файл `docker-compose.yml` и укажите токен вашего бота:
   ```yaml
   environment:
     - BOT_TOKEN=ваш_токен_здесь
   ```
   *(Также вы можете вынести токен в отдельный файл `.env`, если это поддерживается вашей конфигурацией).*

3. Запустите бота в фоновом режиме:
   ```bash
   docker compose up -d
   ```

## Использование
Просто отправьте команду `/start` вашему боту в Telegram. Он сразу ответит, измерит текущую скорость и запустит фоновую задачу для отправки отчетов каждые 10 минут для этого чата.
